## Structure

- `activity`: Activity 폴더
- `fragment`: Activity에서 사용되는 Fragment 폴더
- `media`: 음악 재생과 관련된 클래스 폴더
- `retrofit`: 서버 통신과 관련된 클래스 폴더
- `room`: 데이터 캐싱을 위한 앱 내부 DB와 관련된 클래스 폴더
- `util`: 로그와 앱 테마와 관련된 클래스 폴더
